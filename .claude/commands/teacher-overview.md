# Teacher Overview

전체 학생의 퀴즈 성적을 한눈에 볼 수 있는 **성적 현황판** 컴포넌트를 구현합니다.

## 수행 순서

### 1단계: 현재 소스 파악

아래 파일을 읽는다.
- `hrm-quiz/src/types/index.ts` — `RankRecord` 타입 확인
- `hrm-quiz/src/hooks/useQuiz.ts` — localStorage 키, 랭킹 저장·불러오기 로직
- `hrm-quiz/src/components/` 전체 — 기존 컴포넌트 구조 파악
- `hrm-quiz/src/App.tsx` — 현재 phase 상태 관리 방식 확인

### 2단계: 구현 상태 진단

아래 항목의 현황을 표로 출력하세요.

```
기능                              상태
────────────────────────────────────────────────────
전체 학생 목록 표시               ✅ 구현 / ❌ 미구현
평균 점수 계산 표시               ✅ / ❌
최고 점수 학생 하이라이트          ✅ / ❌
최저 점수 학생 표시               ✅ / ❌
등급 분포 요약 (S/A/B/C/D)       ✅ / ❌
학생 수 표시                     ✅ / ❌
선생님 모드 진입 경로(UI)         ✅ / ❌
```

### 3단계: 성적 현황판 컴포넌트 구현

미구현 항목이 있으면 아래 규격으로 구현하세요.

#### 컴포넌트: `TeacherOverview.tsx`

표시할 정보:
```
┌─────────────────────────────────────────────────┐
│  선생님 모드 — 성적 현황판                         │
│  총 응시자: N명  │  평균 점수: N.N / 20            │
│  최고: {이름} N점  │  최저: {이름} N점              │
├──────────┬──────┬────────┬───────┬──────────────┤
│  이름    │ 점수 │  등급  │  시간  │  응시일시      │
├──────────┼──────┼────────┼───────┼──────────────┤
│  ...     │ ...  │  ...   │  ...  │  ...          │
└──────────┴──────┴────────┴───────┴──────────────┘
등급 분포: S N명  A N명  B N명  C N명  D N명
```

구현 위치:
- `hrm-quiz/src/components/TeacherOverview.tsx` 신규 생성
- 데이터는 `localStorage.getItem('hrm-quiz-rankings')` 로 직접 읽기 (훅 외부)
- 점수 내림차순 → 동점 시 시간 오름차순 정렬 유지

#### App.tsx 연동

- `phase` 타입에 `'teacher'` 추가 (`'start' | 'playing' | 'result' | 'teacher'`)
- StartScreen에 "선생님 모드" 버튼 추가 (비밀번호 없이, 버튼만으로 진입)
- TeacherOverview에 "돌아가기" 버튼 → `phase: 'start'` 로 복귀

### 4단계: 구현 후 검증

구현이 완료되면 아래를 확인하세요.

- `npx tsc --noEmit` 타입 오류 없음
- localStorage 데이터가 없을 때 "아직 응시자가 없습니다." 메시지 표시
- 응시자가 10명 초과일 때 스크롤 처리 (overflow-y-auto)
