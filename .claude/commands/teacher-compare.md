# Teacher Compare

학생 2~5명을 선택해 점수·등급·소요시간을 **나란히 비교**하는 뷰를 구현합니다.

## 수행 순서

### 1단계: 현재 소스 파악

아래 파일을 읽는다.
- `hrm-quiz/src/types/index.ts` — `RankRecord` 타입 확인
- `hrm-quiz/src/components/TeacherOverview.tsx` — 이미 구현되어 있으면 재사용
- `hrm-quiz/src/App.tsx` — phase 및 컴포넌트 연결 방식 확인

### 2단계: 구현 상태 진단

아래 항목의 현황을 표로 출력하세요.

```
기능                              상태
────────────────────────────────────────────────────
학생 선택(체크박스) UI            ✅ 구현 / ❌ 미구현
선택 학생 비교 테이블             ✅ / ❌
점수 차이 강조 표시               ✅ / ❌
1등 학생 하이라이트               ✅ / ❌
비교 초기화 버튼                  ✅ / ❌
```

### 3단계: 비교 뷰 컴포넌트 구현

미구현 항목이 있으면 아래 규격으로 구현하세요.

#### 컴포넌트: `TeacherCompare.tsx`

**학생 선택 영역**
- 전체 학생 목록을 체크박스로 표시
- 최소 2명, 최대 5명까지 선택 가능
- 5명 초과 선택 시 "최대 5명까지 비교 가능합니다" 안내

**비교 테이블 (선택 후 표시)**
```
비교 항목        {학생A}    {학생B}    {학생C}
─────────────────────────────────────────────
점수 (/ 20)        18          14         16
등급               S           B          A
소요 시간         03:21       07:45      05:10
응시 일시        MM/DD HH:mm  MM/DD     MM/DD
순위 (전체)        1위          5위        3위
```

**시각적 강조**
- 각 행에서 최고값 셀: `bg-green-100 font-bold`
- 각 행에서 최저값 셀: `bg-red-50`
- 점수 차이(최고 - 최저)를 테이블 하단에 표시

**구현 위치**
- `hrm-quiz/src/components/TeacherCompare.tsx` 신규 생성
- `TeacherOverview.tsx` 내부에 탭 또는 "비교하기" 버튼으로 전환하여 연동

### 4단계: 구현 후 검증

- `npx tsc --noEmit` 타입 오류 없음
- 학생이 1명 이하일 때 비교 기능 비활성화
- 동일 이름의 학생이 여러 번 응시한 경우, 최고 기록 1건만 사용
