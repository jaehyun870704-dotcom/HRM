# Quiz Range

`hrm-quiz/src/data/questions.ts`에서 id **$1번부터 $2번까지**의 문제를 검토합니다.

## 수행 순서

1. `hrm-quiz/src/data/questions.ts`를 읽는다.
2. `id >= $1` 이고 `id <= $2` 인 문제만 필터링한다.
3. 아래 항목을 순서대로 출력한다.

---

### 문제 목록

각 문제를 다음 형식으로 출력하세요.

```
[id=N] (카테고리 / 난이도 / 출처)
Q. 문제 내용
  ① 보기1  ② 보기2  ③ 보기3  ④ 보기4
  정답: ①~④ 중 하나  |  해설: explanation 첫 문장
```

### 난이도 분포

```
쉬움 N문제 / 보통 N문제 / 어려움 N문제
(어느 한 단계가 전체의 70% 이상이면 "⚠️ 난이도 편중" 출력)
```

### 정답 분포

```
① N문제  ② N문제  ③ N문제  ④ N문제
(편중 경고: 특정 번호가 전체의 50% 이상이면 "⚠️ ①/②/③/④번 정답 편중" 출력)
```

### 카테고리 분포

```
HRM실무 N문제 / 근로관계법 N문제
```

---

해당 범위에 문제가 없으면 `⚠️ id $1~$2 범위에 해당하는 문제가 없습니다.` 를 출력하세요.
