# Teacher Grade Distribution

전체 응시자의 **등급 분포**와 **합격률**을 분석·시각화하는 컴포넌트를 구현합니다.

## 수행 순서

### 1단계: 현재 소스 파악

아래 파일을 읽는다.
- `hrm-quiz/src/hooks/useQuiz.ts` — `calcGrade` 함수의 등급 기준 확인
- `hrm-quiz/src/types/index.ts` — `RankRecord` 타입 확인
- `hrm-quiz/src/components/` — 기존 컴포넌트 현황 파악

### 2단계: 등급 기준 추출 및 출력

`calcGrade` 함수에서 등급 기준을 읽어 아래 형식으로 출력하세요.

```
현재 등급 기준:
  S: 18점 이상 (90%)
  A: 15점 이상 (75%)
  B: 12점 이상 (60%)
  C:  9점 이상 (45%)
  D:  9점 미만

합격 기준: A 이상 (15점 이상) — 조정 필요 시 사용자에게 확인
```

합격 기준이 코드에 명시되지 않은 경우 기본값 `A 이상`을 사용하고 사용자에게 안내하세요.

### 3단계: 등급 분포 컴포넌트 구현

미구현 항목이 있으면 아래 규격으로 구현하세요.

#### 컴포넌트: `TeacherGradeDist.tsx`

**등급 분포 막대 차트 (CSS only, 외부 라이브러리 사용 금지)**
```
S  ████████░░░░░░░░  N명 (XX%)
A  ██████░░░░░░░░░░  N명 (XX%)
B  ████░░░░░░░░░░░░  N명 (XX%)
C  ██░░░░░░░░░░░░░░  N명 (XX%)
D  █░░░░░░░░░░░░░░░  N명 (XX%)
```
- 막대: `div` + Tailwind `w-[XX%]` 인라인 스타일로 비율 표현
- 각 등급별 색상: S=보라, A=파랑, B=초록, C=노랑, D=빨강

**요약 통계**
```
총 응시자:   N명
합격 (A 이상): N명 (XX%)
불합격:       N명 (XX%)
평균 점수:   N.N점
중앙값:      N점
최빈 등급:   {등급}
```

**구현 위치**
- `hrm-quiz/src/components/TeacherGradeDist.tsx` 신규 생성
- `TeacherOverview.tsx` 또는 선생님 모드 탭에서 "등급 분포" 탭으로 연동

### 4단계: 구현 후 검증

- `npx tsc --noEmit` 타입 오류 없음
- 응시자 0명일 때 "데이터 없음" 상태 처리
- 모든 학생이 같은 등급일 때 막대가 100%로 정상 표시되는지 확인
